# Installation Guide

Choose your preferred AI platform to get started with Producer Pal:

## Recommended Options

These are the easiest and most reliable ways to use Producer Pal:

- **[Claude Desktop](./claude-desktop)** - Anthropic's desktop GUI (easiest
  setup)
- **[Producer Pal Chat UI](./chat-ui)** - Built-in chat interface supporting
  Gemini, Mistral, Ollama, and more (free tier available with Gemini)

## Command Line Interfaces

For users comfortable with the terminal:

- **[Gemini CLI](./gemini-cli)** - Google's command line agent (no subscription
  needed)
- **[Codex CLI](./codex-cli)** - OpenAI's command line agent (subscription
  required)
- **[Claude Code](./claude-code)** - Anthropic's command line agent
  (subscription required)

## Web Applications

Use Producer Pal in your browser:

- **[claude.ai Web App](./claude-web)** - Anthropic's web app (requires web
  tunnel)
- **[ChatGPT Web App](./chatgpt-web)** - OpenAI's web app (requires web tunnel)

## Local & Offline Options

Run models completely offline:

- **[LM Studio](./lm-studio)** - Run models offline (experimental, requires
  powerful hardware)
- **[Other MCP-compatible LLMs](./other-mcp)** - Any LLM supporting MCP

## Additional Resources

- **[Upgrading](./upgrading)** - How to update to a new version
- **[Web Tunnels](./web-tunnels)** - Setup remote access (for web apps)
- **[Troubleshooting](./troubleshooting)** - Common issues and solutions

## Requirements

All installation methods require:

- [Ableton Live 12.2+](https://www.ableton.com/live/) with
  [Max for Live](https://www.ableton.com/live/max-for-live/)
- The
  [Producer_Pal.amxd](https://github.com/adamjmurray/producer-pal/releases/latest/download/Producer_Pal.amxd)
  Max for Live device

Additional requirements vary by platform - see the specific installation guide
for your chosen AI.
