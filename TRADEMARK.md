Producer Pal is free software under the GPL-3.0-or-later license. You are free
to fork, modify, and redistribute it under those terms.

### Modification Notices (GPL-3 Section 5a)

The GPL-3 license requires that modified files carry prominent notices stating
that you changed them. Each source file includes a license header — when you
modify a file, add your name to the header:

    // Producer Pal
    // Copyright (C) 2026 Adam Murray
    // Modified by [Your Name]
    // SPDX-License-Identifier: GPL-3.0-or-later

The git commit history serves as the record of what changed and when.

### Rebranding

If you rebrand a fork, you must make clear that it is a modified version and not
the original Producer Pal. Use a different name and do not use the Producer Pal
trademark in a way that suggests endorsement by or association with the original
project.
